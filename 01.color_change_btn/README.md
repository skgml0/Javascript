# 1. 주제
 박스 색깔을 Js로 바꾸는 기능을 구현한다. 
 
# 2. 방법
 script 구문 안에서 색깔 지정을 위한 const 선언
 클릭한 버튼이 해당 const 변수와 일치할 경우, 원하는 속성값으로 변경
 함수화 작업 ( 이전의 canvas 색상을 지우고, 현재 클릭한 색을 추가해서 색상 변경을 할 수 있다.)
